# customer-analysis

This is a simple SaaS revenue analysis tool that allows you to analyze your customer data and see how your revenue is performing

## Metrics

- Monthly Revenue
- MRR
- ARR
- Net Revenue Retention
- Average Contract Value
- Active Customers

## Getting Started

1. Clone the repository
2. Run `npm install`
3. Run `npm run build`

## Developing

1. Clone the repository
2. Run `npm install`
3. Run `npm run dev`
